Changelog:
# POE Part 1

# Project Concept

Established the organisation: Genesis Construction SA.

Defined the primary objective: create an informational and promotional website for a construction company.

Identified content needs for five pages:

Home

About

Products & Services

Gallery

Contact

# 2. Target Audience & Personas

Created initial personas (homeowners, community members, businesses).

Identified user goals (learn about services, view projects, contact company).

# 3. Information Architecture

Developed the first sitemap.

Simple, flat hierarchy with five main menu pages.

# 4. Wireframes

Basic low-fidelity wireframes created.

Early layout concepts without spacing, alignment or interaction notes.

# POE Part 2

# 1. Visual Design

Chose gold + black colour scheme.

Adopted Montserrat as main font.

Built first digital mockups.

# 2. Early HTML Structure

Created initial versions of:

index.html

aboutus.html

services.html

gallery.html

contactus.html

# 3. Usability Adjustments Based on Early Feedback

Improved spacing between menu items.

Adjusted image sizes.

Added hero headings.

Started consistency improvements for layouts.

# 4. Issues Identified in Lecturer Feedback

Inconsistent navigation between pages.

Inline CSS instead of external stylesheet.

Missing alt text for images (accessibility issue).

Gallery layout breaking on smaller screens.

Weak sitemap explanation.

Wireframes lacked descriptions.

No responsiveness on mobile.

Lack of design justification in the report.

# POE Part 3

# 1. Complete Code Rewrite

Rebuilt every page using clean, semantic HTML5.

Moved all styling to a global stylesheet (style.css).

Ensured consistent navigation across all pages.

Added consistent layout spacing, typography and structure.

# 2. Accessibility Improvements

Added alt="" text to all images.

Ensured readable font sizes.

Improved colour contrast.

Ensured all pages use a single <h1> for screen readers.

Improved keyboard-friendly navigation.

# 3. Responsiveness Enhancements

Added responsive CSS grid layouts for services and gallery.

Ensured mobile/tablet scaling using modern CSS techniques.

Standardised image ratios and removed filenames with spaces.

# 4. UI/UX Enhancements

Added hover transitions for images and links.

Standardised headings across all pages.

Improved content hierarchy (H1 → H2 → H3).

Balanced margin and padding for better readability.

Rebuilt hero sections for clarity and visual appeal.

# 5. Contact Page Enhancements

Added Google Maps embed.

Redesigned contact info into stylish information boxes.

Added operation hours using consistent layout patterns.

# 6. Gallery Fixes

Rebuilt gallery using CSS grid.

Implemented responsive image scaling.

Replaced broken images and renamed files.
